<h2 >
  KP Prints – Custom T-Shirt E-Commerce Website

</h2>
KP Prints is an online custom fashion platform that allows users to order personalized T-shirts with their own designs, images, or text. The website provides a simple ordering process where customers select a product and complete the order through WhatsApp communication.

The goal of this project is to create a simple, fast, and accessible custom T-shirt ordering system.

Project Overview

The platform allows customers to:

Browse custom T-shirt products
Click Buy Now to contact the store via WhatsApp
Send their design, size, and delivery details
Review the design preview before printing
Receive a high-quality printed T-shirt delivered within 2–3 business days

All products are printed on premium high-GSM T-shirts using professional printing technology.

<h5> Key Features</h5>
Simple and responsive e-commerce interface
WhatsApp-based ordering system
Custom design submission by customers
Design approval before printing
High-quality T-shirt printing workflow
Fast delivery process
Mobile-friendly user interface
Order Workflow

<h5>The ordering process follows this flow:</h5>

1. User visits the website
2. User selects a product
3. User clicks Buy Now
4. Website redirects user to WhatsApp
5. Customer sends design, size, and address
6. Design preview is created and shared
7. Customer approves the design
8. Payment confirmation
9. Printing on premium high-GSM T-shirt
10. Product shipped and delivered
